AR Navigation

A new method to navigate

Nowadays navigation application are mostly show on a map or some pictures capture by the satellite. It isn’t convenience for the users are finding route to go to the destination or the drivers on the road since they need to care about the road situation and watch the map on the phone or navigator same time. So if the user are walking on the street and using smart phone’s navigation application same time, not all the users can handle two tasks at same time, so they may collide to other passengers or some blocks an occur unnecessary accident.

Date: 2018
Category: Final Year Project




Since it is for the drivers to use it for navigation, so I am trying to let the step to use this app as easy as possible.
